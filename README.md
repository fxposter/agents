# Agents

## Skills

- [`/tmux`](skills/tmux) - simplified version of the Armin's [tmux](https://github.com/mitsuhiko/agent-stuff/blob/main/skills/tmux/SKILL.md) skill.
  `tmux -L agent.sock list-sessions` and `tmux -L agent.sock capture-pane` can be added to allowlists in claude/codex.
